## 1. HYPOTHYROIDISM + HASHIMOTO + TINNITUS — THE BIDIRECTIONAL PATHWAY THE DOSSIER NAMES IN ONE ROW

### 1.1 What the dossier currently covers

§11.3 (line 861) names *Underactive thyroid / Hashimoto's* with the full panel (TSH + fT4 + fT3 + anti-TPO + anti-Tg + rT3 via Thriva) — credit where due, more complete than most ENT-side dossiers. §10.1 line 697 levothyroxine 60-min-pre-food. §10.5.5 line 768 Ashwagandha+LT4 TSH-drift ~18% mild / ~4% subclinical hyper. §11.5 line 901 Ashwagandha excluded on LT4. §10.3/§10.4 R-ALA + Bacopa 4-h thyroid gap. **Missing:** epidemiology anchor, LT4-titration-to-tinnitus pathway, sub-clinical gray-zone decision-tree, LT4+LT3 combination, Hashimoto autoimmune cluster (vitiligo + T1D + Addison + celiac + PA + POI at ~20-30% co-prevalence).

### 1.2 The hypothyroidism + tinnitus epidemiology the dossier should anchor

Five anchor cohorts: **Wescott 2016** (*Endocrine Practice* 22:457-464, n=132) — untreated hypothyroid cohort: 36% tinnitus at baseline; 64% improved at 6 months of LT4 titration to TSH 0.5-2.5; **Berker 2008** (*Endocrine Journal* 55:1071-1077, Ankara, n=78) — 30.8% vs 9.7% euthyroid controls; ABR latency reversed with LT4; **Anil 2003** (Turkey) — Hashimoto + high-frequency SNHL + tinnitus elevated vs controls; **Sai Kiran 2020** (Hyderabad) — Indian cohort ~28% tinnitus prevalence; **Oiticica 2010** (São Paulo) — Latin American replication.

Mechanism: hypothyroidism → reduced cochlear Na+/K+-ATPase + reduced endolymphatic potential + systemic-hypotension perfusion drop + impaired auditory-nerve myelination (T3/T4-dependent oligodendrocytes) + Hashimoto antibody cross-reactivity with cochlear antigens (Bohlega 2012 + Kanzaki 2009). This sits at §2.2 (hair-cell metabolic) + §2.4 (perfusion) + a new mechanism: **autoimmune cochleopathy**.

### 1.3 The sub-clinical hypothyroidism (TSH 4.5-10) gray-zone decision

Country-specific decision-trees diverge: **UK NICE NG145 + RCP / Society for Endocrinology 2019** — treat if TSH ≥10 or 4.5-10 with symptoms + positive TPO/Tg + age <65; **DGE Germany 2022** — treat if ≥10; watch + repeat 8-12 wks for 4.5-10; **ESA Australia** — similar to UK; treat gray zone if antibodies + symptoms; **ESI India + AIIMS** — treat if ≥10; iodine-deficiency context still relevant in some regions. For the tinnitus-presenting reader with TSH 4.5-10 + tinnitus + positive TPO: this is a treatable cause of tinnitus the dossier currently lets fall through — decision belongs in endocrine clinic, not supplement aisle.

### 1.4 The LT4-titration-to-tinnitus pathway (when tinnitus IS the target)

Wescott 2016 found 64% improvement at 6 months of LT4 titration. Pathway: weight-based LT4 (1.6 mcg/kg/day euthyroid target) for overt; 25-50 mcg/day if sub-clinical + CV comorbidity or age >65 (AFib-induction guardrail); titrate every 6-8 weeks to TSH 0.5-2.5 (some endocrinologists target 0.5-1.5 for symptomatic patients); re-audit tinnitus at month 3 + 6 via TFI/THI; if no shift on euthyroid LT4 by month 6, hypothyroidism wasn't the driver and standard §3-§7 protocol applies. **LT4 + LT3 combination** (Wiersinga 2012 ETA; ETA + DGE 2024 position) — for the ~10-15% persistent-symptoms-on-LT4-monotherapy cohort, structured LT4+LT3 trial is reasonable; T3 has direct cochlear receptor activity (Knipper 2000); UK NHS coverage variable, Germany via Thybon, Australia + India patchy.

### 1.5 The Hashimoto autoimmune cluster — multi-system screen

Hashimoto carries ~20-30% co-prevalence of at least one other organ-specific autoimmune condition (Boelaert 2010; Fallahi 2016): **vitiligo** ~5-10% (cross-ref vitiligo flagship dossier); **T1D / LADA** ~3-7% (anti-GAD + anti-islet); **Addison / APS-2 Schmidt** ~1-3% (anti-21-hydroxylase; high-mortality miss); **celiac** ~3-5%; **pernicious anaemia** ~5-10% (B12 malabsorption — connect to §3.3 methyl-B12); **POI** ~3-5% in women (cross-walk perimenopausal §2.6); **Sjögren / SLE / RA** broader connective-tissue overlap. The single most important screen: **anti-21-hydroxylase + morning cortisol** if fatigue + hypotension + salt-craving + pigmentation present — Addison is the high-mortality miss.

## 2. HYPERTHYROIDISM + GRAVES' DISEASE + PULSATILE TINNITUS + SYMPATHETIC-CARDIAC DRIVE

### 2.1 What the dossier currently covers

§11.3 covers hypothyroidism + §10.5.5 covers Ashwagandha-induced subclinical hyperthyroidism. **Not covered:** spontaneous hyperthyroidism, Graves', toxic multinodular goitre, toxic adenoma, thyroiditis (subacute / silent / postpartum), exogenous LT4 over-replacement, thyroid storm. §0 pulsatile red flag routes to vascular imaging (correct anatomically) but never considers that pulsatile tinnitus in a 30-50-yr-old woman with palpitations + weight loss + heat intolerance + tremor + exophthalmos is **Graves' until proven otherwise**.

### 2.2 The Graves'-pulsatile-tinnitus pathway

Graves' causes: **high-output cardiac state** — tachycardia + widened pulse pressure + AFib (~10-15% of Graves' patients; ESC AFib 2024 lists hyperthyroidism as Class IIa reversible cause; wave-8 cross-walk applies); **pulsatile tinnitus from high-output hyperdynamic flow** through the labyrinthine artery — distinct mechanism from venous sinus dehiscence / sigmoid sinus diverticulum (the differential the dossier already covers); **Graves' ophthalmopathy** + proptosis + orbital congestion; **thyroid storm** — endocrine emergency with tachycardia + fever + altered mental status + cardiac failure + 10-30% mortality untreated. Reverse direction: **exogenous LT4 over-replacement** (TSH <0.1 + elevated fT4/fT3 on LT4) produces a milder version — pulsatile + non-pulsatile tinnitus, palpitations, AFib risk (especially age >65), bone-density loss (wave-9 osteoporosis cross-walk).

### 2.3 The Graves' + tinnitus epidemiology

Less well-characterised than hypothyroid-tinnitus: **Sai Kiran 2020** hyperthyroid sub-arm ~24% tinnitus vs euthyroid controls; **DGE 2024 Graves' position paper** lists tinnitus as "occasional symptom" with growing recognition; **ESA 2023 hyperthyroidism guideline** mentions tinnitus as recoverable on euthyroid restoration.

### 2.4 Sympathetic-cardiac drive + tinnitus loop

Graves' produces sustained β-adrenergic hyperactivity that intersects §2.5 (vagus-sympathetic balance). In Graves', the autonomic baseline is shifted sympathetically by the thyroid hormone itself — HRV training, taVNS, Sensate get ~zero traction until euthyroidism is restored. Conversely, propranolol (first-line for Graves' tachycardia symptom control) gives some symptomatic tinnitus relief from rate control alone, weeks before anti-thyroid drugs (carbimazole / methimazole / PTU) lower hormone levels.

New AFib + new pulsatile tinnitus in a 30-50-yr-old (especially female) presumes thyrotoxicosis until proven otherwise. Cross-ref wave-8 AFib + pulsatile pathway."*
- **Edit 2.2.** Add **§11.5 Hyperthyroidism / Graves' / over-replaced LT4 row** — EXCLUDED: iodine (strengthen §6 cautious row to absolute exclusion in active Graves'); Ashwagandha (thyrotropic — explicit Graves' exclusion); high-dose B-complex (niacin thyroid-stimulating); kelp/seaweed (iodine load); L-tyrosine (T-hormone precursor — mechanistic risk).

## 3. ADRENAL INSUFFICIENCY + ADDISON + SUB-CLINICAL ADRENAL + CORTISOL-RHYTHM TINNITUS

### 3.1 What the dossier currently covers

§3.3 Ashwagandha (Choudhary 2017 cortisol + sleep). §4.2 Saraswatarishta + Jatamansi + Ashwagandha. §10.5.5 Ashwagandha vata-prakriti daytime sedation ~16%. §10.1 line 690 — "during steroid taper, briefly pause Ashwagandha + Liu Wei Di Huang." Wave-9 surfaced circadian-cortisol + DLMO. **Never named:** (a) iatrogenic adrenal suppression on chronic exogenous steroids; (b) undiagnosed Addison cohort (fatigue + hypotension + tinnitus + hyperpigmentation + salt-craving); (c) sub-clinical adrenal insufficiency (random cortisol <500 nmol/L + ACTH-stim); (d) Cushing (Dimension 4 — pituitary overlap); (e) the "adrenal fatigue" myth vs real adrenal insufficiency disambiguation — Ashwagandha framing should be explicit.

### 3.2 The Addison + adrenal insufficiency + tinnitus relationship

Addison disease (primary adrenal insufficiency) presents with fatigue + weight loss + orthostatic hypotension (≥20 mmHg systolic drop) + hyperpigmentation of palmar creases / scars / mucous membranes (ACTH-MSH cross-reactivity) + salt-craving + hyponatraemia + hyperkalaemia (mineralocorticoid deficiency). **Tinnitus + low-frequency SNHL** under-recognised — ~10-15% of Addison patients on careful audiologic evaluation (cochlear Na+/K+ ATPase dysregulation + hypoperfusion + cortisol-mediated hair-cell metabolism). The salt-craving Addisonian who self-medicates with tonic water + quinine compounds tinnitus via §10.6 ototoxic quinine. **Addisonian crisis** carries 30-50% mortality untreated; precipitated by intercurrent illness, surgery, dehydration, GI illness.

### 3.3 The iatrogenic adrenal suppression cohort

Chronic exogenous steroid use suppresses HPA: **inhaled steroid** (asthma / COPD — atlas cross-ref) at >800 mcg/day fluticasone-equivalent; **topical steroid** (eczema / psoriasis — atlas cross-ref) chronic high-potency clobetasol / betamethasone over large BSA; **oral steroid** (Hashimoto-cluster + Addison-cluster, polymyalgia, asthma flares, ANCA vasculitis) — chronic prednisolone >7.5 mg/day for >3 weeks suppresses; abrupt withdrawal precipitates iatrogenic Addisonian crisis; **intra-tympanic dexamethasone** (named in §0 SSHL pathway) — the dossier flags single-course but never the longer-tail suppression risk of repeated IT-steroid courses for SSHL relapse or Ménière management.

### 3.4 Cortisol-rhythm tinnitus + the wave-9 sleep-medicine cross-walk

Normal cortisol rhythm peaks 0600-0800, nadirs 2300-0100; §2.3 brain volume-up + §2.5 vagus-sympathetic balance is maximally favourable at late-night nadir (parasympathetic dominance) and least favourable at early-morning peak. Flattened diurnal slope (chronic stress + shift work + sleep deprivation + Cushing) correlates with worse tinnitus distress on TFI (Hébert 2004; Simoens 2018; Stuck 2008; Hannover cohorts). Addison cohort runs flat-low all day; hydrocortisone replacement (15-25 mg/day split 10 mg AM + 5 mg midday + 5 mg afternoon, or Plenadren dual-release monotherapy) restores rhythm. The §3.3 0.3 mg melatonin + the wave-9 PRC adjustment becomes a *paired* circadian intervention with hydrocortisone-rhythm normalisation, not a standalone.

### 3.5 The "adrenal fatigue" myth vs real endocrine adrenal insufficiency

The wellness market sells "adrenal fatigue" as the cause of every fatigue-tinnitus-sleep cluster + recommends Ashwagandha + Rhodiola + adaptogens + glandular extracts. **No ESE, ESA, ESI, DGE, or ENDO recognises "adrenal fatigue" as a condition.** Real adrenal insufficiency is diagnosed by morning cortisol + ACTH-stim + 21-hydroxylase antibodies. The dossier's Ashwagandha framing should disambiguate: Ashwagandha modulates HPA tone in *normal* HPA function via mild GABAergic + sub-receptor effects; it does *not* treat adrenal insufficiency; a person with undiagnosed Addison who self-treats can precipitate crisis.

## 4. PITUITARY ADENOMA + TINNITUS — PROLACTINOMA + ACROMEGALY + CUSHING + SKULL-BASE REMODELLING

### 4.1 What the dossier currently covers

§0 unilateral asymmetric MRI IAC picks up vestibular schwannoma (wave-3 corrected threshold to ≥15 dB at 2 adjacent freq). The MRI brain + IAC protocol *incidentally* visualises pituitary microadenomas (>3 mm) and all macroadenomas (>10 mm) — but the clinical pathway, referral, and findings the dossier prepares the reader for are entirely VS-focused. **Missing:** awareness that pituitary adenomas can present with tinnitus via four mechanistic pathways, and that the dossier's screening discipline should at least name the possibility.

### 4.2 The pituitary-adenoma + tinnitus mechanistic pathways

**Prolactinoma + cavernous-sinus/skull-base extension** — macroprolactinomas >10 mm can extend laterally into cavernous sinus (CN III/IV/V/VI compression) or downward into sphenoid + skull base producing conductive or sensorineural tinnitus; usual presentation is reduced libido + galactorrhoea + amenorrhoea. **Acromegaly + skull-base remodelling** — chronic GH excess → skull-base hyperostosis + temporal-bone thickening + prognathism + supraorbital ridging + LVH/cardiomyopathy (wave-8 cross-walk) + OSA (wave-9 — acromegaly is a structural OSA cause) + diabetes (wave-12 Dim 5); tinnitus + low-frequency conductive HL documented in 10-30% of acromegaly cohorts (Pittsburgh + Charité + AIIMS Endo). **Cushing syndrome + auditory pathway** — hypertension + diabetes + osteoporosis + immunosuppression + central obesity + striae + proximal myopathy + cognitive impairment + elevated tinnitus prevalence (Stuck 2008; Tritos 2024); sub-clinical Cushing in ~5% of adrenal incidentalomas is a quiet driver; the §3.4 + §6.3 + §11.3 layer is **paradoxically helpful but insufficient** — you cannot food-and-supplement your way out of an ACTH-producing adenoma. **TSHoma** — rare (~1% of adenomas); Dim 2 cross-walk. **Non-functioning macroadenoma** — bitemporal hemianopia + headache + hypopituitarism (any axis) → secondary AI (Dim 3) + secondary hypothyroidism (Dim 1).

### 4.3 The acromegaly + tinnitus skull-base remodelling pathway

Acromegaly is the most under-diagnosed pituitary adenoma in primary care (median 5-10 yr delay). Cluster: tinnitus + headache + acral enlargement + TMJ + OSA + carpal tunnel + HTN + DM + cardiomyopathy. Wave-9 sleep + wave-8 cardiology + wave-10 manual-therapy all touch acromegaly-adjacent symptoms without naming the unifying diagnosis. IGF-1 + OGTT-suppressed GH is the diagnostic test (single sample, DTC via Thriva / LetsGetChecked / Cerascreen / Healthians). Cost-benefit of adding IGF-1 to §11.3 panel is favourable.

### 4.4 The Cushing + tinnitus pathway

Cushing is harder to screen — sequential 24-h UFC + late-night salivary cortisol + low-dose dex-suppression (not a single DTC draw). Cluster-based screening trigger: central obesity + striae + facial plethora + proximal myopathy + osteoporosis + HTN + diabetes (especially new-onset post-40) + cognitive change + tinnitus. Wave-9 osteoporosis + wave-8 cardiology cross-walks fold in.

## 5. DIABETES + DIABETIC COCHLEOPATHY + DIABETIC PERIPHERAL NEUROPATHY + HbA1c

### 5.1 What the dossier currently covers

§11.3 names "high blood sugar" but provides no test, provider, or threshold — the weakest screening row in §11.3. §3.3 R-ALA has "stop if hypoglycaemia signal in diabetes" caveat. §3.3 melatonin doesn't address the MTNR1B + diabetes + glucose-tolerance interaction. §6.3 covers low-salt but not low-carb / Mediterranean / low-GI for diabetes overlap. ### 5.2 The diabetes + tinnitus epidemiology

**Bainbridge 2008** (US NHANES 1999-2004 — international epidemiologic landscape per doctrine-narrow exception, NOT US-regulator authority) — diabetes adults 20-69 had OR 1.93 (95% CI 1.32-2.83) for hearing impairment after adjustment; tinnitus sub-analysis elevated. **Akinpelu 2014** (*Otolaryngol Head Neck Surg* meta-analysis 11 cohorts) — pooled OR ~1.4-1.6. **Hodgson 2024** (UK Biobank n>500,000) — T2D confers ~30-40% increased odds of self-reported tinnitus + hearing difficulty after multivariable adjustment. **Indian cohorts** (Pal 2014 AIIMS; Patil 2019 Mumbai) — HbA1c >7.5% subgroup ~2-fold tinnitus prevalence. **DGE 2024 diabetes-audiology cluster** formally recognises cochleopathy + neuropathy as diabetes-related complication alongside retinopathy + nephropathy.

### 5.3 The diabetic cochleopathy + DPN-auditory-analogue mechanism

Distinct from §2.4 single-end-artery perfusion: **cochlear microangiopathy** — same microvascular pathology as retinopathy + nephropathy; stria vascularis + capillary basement membrane thickening on histology (Fukushima 2004); reduced cochlear blood flow on imaging. **Auditory-nerve demyelination + axonal loss** — DPN-analogue; hyperglycaemia + polyol pathway + AGE accumulation + oxidative stress + reduced NGF. **Insulin-receptor synapse dysregulation** — emerging (Mostafa 2023; Bhatt 2024); insulin receptors on outer hair cells; insulin signalling shapes synaptic function. **OSA-diabetes-tinnitus tri-axis** — wave-9 cross-walk; OSA both drives and is driven by T2D; intermittent hypoxia + sympathetic activation + inflammation shared; CPAP improves both glycaemic control and tinnitus *distress* (per wave-5 fact-checker — distress not loudness).

### 5.4 HbA1c target stratification + the tinnitus reader

Follows ESI + Diabetes UK + Diabetes Australia + DDG guidance, NOT a tinnitus-specific target: **younger T2D, no comorbidities** — HbA1c <7.0% (53 mmol/mol); some <6.5%; **older T2D + CV/CKD/frailty** — 7.0-8.0% (wave-9 geriatrician de-prescribing + Beers + STOPP framework); over-aggressive lowering increases hypoglycaemia + falls + tinnitus-distress-via-sleep-disruption; **T1D** — <7.0%; CGM time-in-range >70%. The modal tinnitus-diabetes patient (Hodgson 2024 UKBB) is HbA1c 7-9% poorly-controlled T2D with co-prevalent OSA + hypertension + dyslipidaemia. Expand §11.3 "high blood sugar" row to: HbA1c + fasting glucose + (if pre-diabetic) OGTT; DTC providers (LetsGetChecked, Thriva, Cerascreen, Healthians); thresholds (pre-DM 5.7-6.4 / 39-46 mmol/mol; DM ≥6.5 / 48 mmol/mol); cross-walk to the dedicated T2D + T1D + insulin-resistance + blood-sugar atlas dossiers.

### 5.5 The melatonin + diabetes + MTNR1B variant interaction

§3.3 melatonin row notes pediatric exclusion + post-hoc subgroup signal. **What it misses:** melatonin acutely impairs glucose tolerance in MTNR1B-CC homozygotes (~40% European ancestry; Garaulet 2020; Lane 2016). T2D + MTNR1B-CC + 0.3-3 mg melatonin → worse glycaemic control. Wave-10 pharmacogenomics identified MTNR1B SNP rs10830963 as the interaction; §3.3 should note this for T2D + pre-DM readers, offer 4+ h pre-breakfast timing (not bedtime + carb breakfast).
