## 1. WCAG 2.2 AA + EAA 2025 WEB-ACCESSIBILITY AUDIT OF THE LIVE RENDER

### 1.1 What's being audited

The dossier is consumed on a website. Wave-3 disability-inclusion audited the *content* surface (non-auditory analogues, Deaf-identity framing, CI contraindications, signed-language-validated instruments). Wave-13 audits the *rendering* surface — `https://nichecore-tytn7sfqja-uc.a.run.app/brain/tinnitus-support` rendered by `app/[domain]/[slug]/page.tsx` inside `components/layout/dossier-shell.tsx` with `LeftRailNav` + `MindMapStrip` + `DeepDive` collapsibles + the in-progress `DecisionHero` card grid + footer + mobile dock/tab-strip + global theme tokens in `app/globals.css`. The applicable accessibility standards on the WCAG 2.2 AA Level minimum:

- **WCAG 2.2 (W3C Rec October 2023)** — Levels A + AA + AAA. AA is the regulatory floor in UK / AU / EU / many states. WCAG 2.2 introduced 9 new Success Criteria (2.4.11 + 2.4.12 + 2.4.13 Focus Not Obscured; 2.5.7 Dragging Movements; 2.5.8 Target Size Minimum; 3.2.6 Consistent Help; 3.3.7 Redundant Entry; 3.3.8 + 3.3.9 Accessible Authentication) beyond WCAG 2.1; some are AA-mandatory now.
- **EN 301 549 V3.2.1 (2021)** — the EU harmonised accessibility standard; references WCAG 2.1 + adds non-web ICT requirements.
- **EAA 2019/882** (Directive, applicable from 28 June 2025 across EU + EEA + UK-Northern-Ireland-Protocol relevant) — e-commerce + consumer banking + e-books + ticketing + transport-info-services + telephony services + audiovisual-media-services. NicheCore's storefront sits in scope as e-commerce.
- **UK Public Sector Bodies (Websites and Mobile Applications) Accessibility Regulations 2018** — applies to PSBs; private NicheCore is not directly bound but Equality Act 2010 s. 20 reasonable-adjustments duty applies to service-provider websites under existing case-law (RNIB v BMI Baby 2012 settlement).
- **AU DDA 1992 s. 5 + 24** — read by HRC + Federal Court as covering websites (Maguire v SOCOG 2000 Sydney Olympics ruling; reinforced 2022 DDA Inquiry).
- **DE BGG + BITV 2.0 (Barrierefreie-Informationstechnik-Verordnung 2011, revised 2019)** — applies federal sites; private subject to AGG s. 19 service-discrimination + BGG-future-expansion.
- **IN RPwD Act 2016 s. 40-42** — accessibility standards for ICT mandatory; DEPwD's Guidelines for Indian Government Websites 3.0 (GIGW 3.0 2023, WCAG 2.1 AA-aligned).

### 1.2 Predicted failure modes — component-by-component (this is an architecture-audit absent live-DOM-inspection but anchored in the component code we know exists)

**Failure mode 1.2.a — DeepDive collapsibles (`components/layout/deep-dive.tsx`).** The "#### Open the science" pattern wraps every deeper-layer block inside a collapsible. Likely WCAG failures: (i) **SC 1.3.1 Info and Relationships** if the disclosure widget uses a `<div>` with `onClick` rather than `<button aria-expanded>`; screen-reader announces "clickable, Open the science" not "button collapsed, Open the science"; (ii) **SC 2.1.1 Keyboard** if `Enter` toggles but `Space` does not (button behaviour requires both); (iii) **SC 4.1.2 Name, Role, Value** if `aria-controls` doesn't point to the panel ID; (iv) **SC 1.4.13 Content on Hover/Focus** if hover-reveal is used anywhere within (must be dismissable + hoverable + persistent). Fix: enforce `<button type="button" aria-expanded aria-controls={panelId}>` + matching `<div id={panelId} role="region" aria-labelledby={triggerId}>`.

**Failure mode 1.2.b — LeftRailNav tree (`components/nav/left-rail-nav.tsx`).** An expandable H2 + H3 tree. Likely failures: (i) **SC 4.1.2** if not implemented as `role="tree"` + `role="treeitem"` + `aria-expanded` + `aria-level` per WAI-ARIA Authoring Practices 1.2 Tree pattern; screen-readers (NVDA + JAWS + VoiceOver + TalkBack) will read it as a flat list; (ii) **SC 2.1.1 Keyboard** if Left/Right arrow doesn't collapse/expand + Up/Down doesn't move between treeitems + Home/End doesn't jump to first/last (tree-widget keyboard contract); (iii) **SC 1.4.11 Non-text Contrast** if the chevron indicator is < 3:1 against background.

**Failure mode 1.2.c — MindMapStrip sticky horizontal chip-row (`components/nav/mind-map-strip.tsx`).** Sticky horizontal flex-wrap. Likely failures: (i) **SC 1.4.10 Reflow** if at 320 CSS-px width (smallest mobile) the strip overflows or pushes content; reflow requires no horizontal scroll except for data-tables/maps/images; (ii) **SC 2.4.11 Focus Not Obscured (Minimum) — new in WCAG 2.2** if a focused chip is hidden behind sticky chrome; (iii) **SC 1.4.12 Text Spacing** if increased letter-spacing/word-spacing breaks the chip layout.

**Failure mode 1.2.d — DecisionHero card grid (in-progress, `components/layout/decision-hero.tsx`).** Founder's above-fold 6-card decision-fast surface. Predicted: (i) **SC 2.5.8 Target Size (Minimum) — new in WCAG 2.2 AA** requiring 24×24 CSS-px minimum click target (cards usually pass; risk is the within-card "Open" affordance); (ii) **SC 1.3.2 Meaningful Sequence** — visual grid order must match DOM order for screen-readers; (iii) **SC 3.2.4 Consistent Identification** — if "Quick start" appears as a card on the home page + as a section header on the dossier + as a label inside the deep-dive, the identification must be consistent.

**Failure mode 1.2.e — GradeBadge colour-coding (`components/ui.tsx`).** GradeBadge A/B/C/D/T/H/X uses colour. Likely failure: **SC 1.4.1 Use of Colour** — if grade conveyed only by hue (green/amber/red) without a text label or shape/icon, fails. Also **SC 1.4.3 Contrast (Minimum)** for the grade letter against the badge background must be ≥ 4.5:1. Tailwind v4 + the `ACCENTS` map in `lib/evidence.ts` need explicit contrast tokens.

**Failure mode 1.2.f — FreshnessChip + RiskTag pills.** Same colour-as-only-signifier risk as GradeBadge.

**Failure mode 1.2.g — Dossier-shell back-link `← Atlas`.** Wave-3 specified clickable brand → home + back-link → atlas + GO TO site-nav. Predicted failures: (i) **SC 2.4.1 Bypass Blocks** — skip-link to main content required; need `<a href="#main" class="sr-only focus:not-sr-only">Skip to main content</a>` as first focusable element; (ii) **SC 2.4.4 Link Purpose (In Context)** — `← Atlas` is OK in context but screen-reader-list view sees only "Atlas" — preferable `aria-label="Back to atlas index of all journeys"`.

**Failure mode 1.2.h — Tables in §3.3, §4.2-4.7, §10.1, §13.7.** Predicted: (i) **SC 1.3.1** if `<table>` lacks `<th scope="col">` / `<th scope="row">`; (ii) **SC 1.3.2 Meaningful Sequence** — multi-column tables read column-by-column by NVDA must still convey row-level meaning (the TL;DR column added in v6.2.1 needs to be reachable as a row-header in screen-reader-table-mode); (iii) **SC 1.4.10 Reflow** — long tables horizontal-scroll on mobile; provide a stacked-card mobile alternative or expose the TL;DR-only summary as the mobile-default with table-on-demand.

**Failure mode 1.2.i — Search shortcut ⌘K / Ctrl-K.** WCAG 2.1.4 Character Key Shortcuts requires single-key shortcuts to be remappable or only-active-on-focus. ⌘K + Ctrl-K being modifier-combos are fine; verify no single-letter (e.g. J/K/PgUp from `use-keyboard-nav.ts`) shortcut fires globally without focus — that would fail SC 2.1.4.

**Failure mode 1.2.j — Crisis-line block at the top.** WCAG 1.3.1 + 1.3.5: the crisis-line phone numbers should be `tel:` links so screen-reader users on mobile can one-tap dial. SignHealth BSL routing should be a `<a>` to the SignHealth website, not unlinked text. Same for 988 video-relay.

### 2.2 Benchmarks the dossier should be hitting

- **WCAG 3.1.5 Reading Level (AAA, optional)** — if text requires reading ability more advanced than the lower-secondary level, provide a supplemental version. Lower-secondary = ~Grade 9. WCAG 2.2 is moving 3.1.5 from AAA-optional to AA-soft-recommendation in WCAG 3.0 draft.
- **UK GDS Plain English style guide** — target Reading Age 9 for public-facing content; the NHS health-information style guide uses Reading Age 11.
- **PLAIN (Plain Language Association International)** — clearest meaning + active voice + short sentences + everyday words.
- **EU Plain Language Initiative + Council Resolution on plain language in EU acts**.
- **DIN 8581-1:2023 Leichte Sprache** (German Easy Language) — sentences ≤ 8 words ideal; one main clause per sentence; no passive; common everyday vocabulary; pictograms paired.
- **AU Government Style Manual + AHRC Plain English Guide**.
- **IN Plain Language Initiative + RPwD-Act-§42 accessible information mandate**.
- **WHO HEARING report 2021 + WHO TIP toolkit** — sets plain-language framing for hearing-health public-facing content; explicit "person-first or identity-first per reader preference" framing.
- **CHANGE People Easy Read** (UK) + **Hablamos Junto** (US Latinx health-literacy, OK as comparator) + **Plain Cymraeg** (Welsh) — established easy-read providers; their style sheets are public.

### 2.3 The cognitive-accessibility layer beyond reading-grade

Reading-grade is necessary but insufficient for cognitive accessibility. Additional barriers in the current dossier:

- **Length.** ~18,000 words is roughly 4-6 hours of reading at Grade-15. Cognitive-disability readers (ID + acquired brain injury + dementia + ADHD + autistic + dyslexic + post-stroke aphasia + ME/CFS brain-fog + chemo-brain post-cancer + long-COVID brain-fog) cannot sustain 4-6h of focus. ## 3. AUDISM + AUDIST-FRAMING AUDIT

### 3.1 What audism is, and why wave-3 didn't reach it

Audism (Humphries 1977, dissertation; re-developed Bauman 2004 *JDSDE*; expanded Bauman & Murray 2014 *Deaf Gain*; explored Lane 1992 *The Mask of Benevolence*) is the assumption that hearing is a superior mode of being + the institutional + interpersonal practices that flow from that assumption. It includes:

- **Phonocentrism** — privileging spoken-language modalities over signed-language modalities.
- **Hearing-as-default** — designing services for hearing users + treating Deaf accommodation as an exception or after-thought.
- **Cure-orientation** — framing deafness as a deficit to be remediated rather than a difference to be lived with.
- **Pathologising signed-language acquisition** — treating ASL/BSL/Auslan/ISL/LSF/DGS as compensations rather than full natural languages.
- **CI-default for deaf children** — assuming cochlear implantation is the obvious choice rather than one option among many (Sparrow 2005 *AJOB* + Bauman 2008 *Open Your Eyes* + the international Deaf-cultural critique).

Wave-3 disability-inclusion surfaced the **content** gap (no §1.4 Deaf-identity framing, no §5.20 non-auditory protocol, no CI-contraindication block, no signed-language-validated instruments, no Deaf-led MBCT-T variants). That work closed the door against the most flagrant audism. Wave-13 surfaces the residual **epistemic** audism the wave-3 fix did not touch: the dossier still assumes hearing is the goal + restoration the success-state + tinnitus-elimination the desired outcome. It does not honour the Deaf-cultural position that tinnitus, for some Deaf adults, is one feature of being-in-the-world that need not be "fixed" — and that fixing it is no more imperative than fixing left-handedness.

### 3.2 Where residual audism shows in the dossier

- **§1.1 frame.** "Long-term ringing in the ears is rarely 'just nerve damage' — it has four causes you can work on at home." The framing is action-on-cause; success = ringing-reduced. Reframe to: "Long-term ringing in the ears has four causes. **Whether you want to reduce it, learn to live alongside it, or both is your choice** — this protocol gives you the levers for both."
- **§5.9 "OTC hearing aid".** Framed as masking + amplification — the audiologic default. Missing: the Deaf-community framing that hearing-aids-for-tinnitus is one option among many, and that some Deaf-and-tinnitus-experiencing adults reject amplification entirely in favour of cochlear-rest + signed-environment design + vibrotactile enrichment (Bahan 2008 sensorial-orientation framework).
- **§15.4 CI block.** Wave-3 already corrected the over-claim ("CI can resolve"). Wave-13 adds: the Deaf-community CI critique — particularly the paediatric CI debate (the 1990s-2000s + ongoing) where Deaf-community organisations (NAD US, BDA UK, FAD India, Deafness Forum AU) hold that imposing CI on a Deaf child before they can consent + before they acquire signed-language is an audist act. CI-for-an-adult-with-acquired-deafness-who-elects-it is uncomplicatedly the user's choice; CI-as-default-for-paediatric-tinnitus is contested. The dossier never names the contest.
- **§5.2 sound enrichment cornerstone.** Wave-3 added the §5.20 non-auditory analogue. Wave-13 adds: the **decision-making framing** — for a culturally Deaf adult experiencing tinnitus, the **§5.20 vibrotactile path is the cornerstone, not the substitute**; the §5.2 sound-enrichment path is the optional supplement, not the default. The wave-3 §1.2 Quick-start rewrite ("If you are culturally Deaf … vibrotactile tonight + Deaf-led mindfulness this week") gets close; wave-13 extends it through the whole protocol.
- **Goal-statement framing.** §1.3 + §13.5 imply success = TFI reduction + loudness slider reduction. For some Deaf-and-tinnitus-experiencing readers, success = distress reduction + acceptance + integration — and **loudness reduction is irrelevant**. The success criteria must accommodate both goals.

### 3.3 The Deaf-cultural perspective the dossier should honour

The Deaf-cultural position (NAD US + BDA UK + FAD India + Deafness Forum AU + WFD World Federation of the Deaf Helsinki + EUD European Union of the Deaf Brussels):

- **Deafness is a linguistic + cultural identity, not a medical condition.** Signed-language users are members of a language community; identifying Deaf with a capital-D signals cultural-Deaf identity (Padden & Humphries 1988 *Deaf in America*).
- **Deaf Gain (Bauman & Murray 2014)** — the framework that Deaf experience offers distinctive cognitive + perceptual + community-organising gifts; deafness is not loss but difference + addition. Architecture (Gallaudet DeafSpace project), pedagogy (visual-spatial reasoning research), community-organising (deep-democracy + visual-attention norms) all flow from this.
- **Tinnitus-in-Deaf-bodies.** For some pre-lingually Deaf adults, tinnitus is not "ringing" — it's a vibrational or pressure experience (wave-3 detail); the *cure* framing is a category error.
- **CI-decision-making.** A Deaf adult who freely chooses CI is uncomplicatedly exercising autonomy. A paediatric CI-decision made by hearing parents on a pre-lingually-Deaf child is contested by Deaf-community organisations on autonomy + consent + linguistic-deprivation grounds (Lane 1992; Sparrow 2005; Humphries 2017 *Lancet Child Adolesc Health*).

### 3.4 Recommended edits — dimension 3

- **Edit 3.1.** Reframe §1.1 to surface goal-choice — "Whether you want to reduce it, learn to live alongside it, or both is your choice. This protocol gives you the levers for both."
- **Edit 3.2.** Add §1.5 "On hearing-cure framing" — 100 words honouring the Deaf-cultural perspective that hearing is one mode of being, not the goal of being. Cite Bauman & Murray 2014 *Deaf Gain* + Lane 1992 + Padden & Humphries 1988 + NAD/BDA/FAD/Deafness Forum AU position statements.
- **Edit 3.3.** Reframe §13.5 patient-facing primer to surface two success-criteria paths — (a) the loudness/annoyance-reduction path for the reader whose goal is symptom-reduction; (b) the acceptance + integration path for the reader whose goal is wholeness-with-tinnitus. Both are valid; the protocol scaffolds both.
- **Edit 3.4.** Add §15.4b — "Deaf-community perspective on CI" — 200 words surfacing the paediatric-CI contest + the international Deaf-organisation position statements + the principle that the dossier serves users + their goals, not a presumed "restore hearing" objective. Frame as honest education, not editorialising. Cite Lane 1992 *Mask of Benevolence* + Bauman 2008 *Open Your Eyes* + Sparrow 2005 *AJOB* + Humphries 2017 *Lancet Child Adolesc Health* + NAD + BDA + FAD + Deafness Forum AU + WFD + EUD position statements.
- **Edit 3.5.** Apply the same goal-choice frame to every §5.x intervention — each opens with "If your goal is loudness-reduction, this layer helps via X. If your goal is acceptance + integration, this layer helps via Y." Some interventions serve both; some serve one only. Be honest.
- **Edit 3.6.** Add a §0.3 "person-first OR identity-first" note — many Deaf-community readers prefer identity-first language ("Deaf person", "Deaf reader") over person-first ("person who is Deaf"). The dossier honours both per reader preference; per WHO HEARING-report and NAD/BDA style.

## 5. PER-JURISDICTION DISABILITY-BENEFIT + ACCOMMODATION-PATHWAY ADDENDUM

### 5.1 What §15 currently covers

§15 routes plateau-cases to: ENT + audiology (§15.2 Lenire, §15.3 implanted VNS), country-specific reimbursement (§15.4-15.8 — India PMJAY-AYUSH, UK NHS, German GKV, Australian Medicare), and wave-12 chronic-pain-clinic (§15.9). **Disability-benefit + statutory-accommodation pathways are entirely absent**. Severely-distressed tinnitus + comorbid sensorineural-hearing-loss patients qualify for these benefits in non-trivial numbers, and the dossier currently fails to surface them.

### 5.2 UK — PIP + Access to Work + DSA + Equality Act 2010 reasonable adjustments

- **Personal Independence Payment (PIP)** — DWP non-means-tested daily-living + mobility benefit; eligibility is via functional-impact assessment; severe tinnitus + hearing-loss commonly scores points under "Engaging with other people face-to-face" (Activity 9) + "Communicating verbally" (Activity 7) + "Reading and understanding signs, symbols and words" (Activity 8). PIP assessment guidance recognises hearing-loss + tinnitus + sound-sensitivity-driven hyperacusis. Citizens Advice + Disability Rights UK + RNID publish PIP-application support guides.
- **Access to Work** — DWP scheme reimbursing employer accommodations for disabled workers — funds noise-cancellation headphones + assistive-listening devices + sound-masking + communication-support-worker + signed-language-interpreter + ergonomic-quiet-workspace adjustments. Severely tinnitus-affected employees routinely qualify.
- **Disabled Students' Allowance (DSA)** — Student Finance England + equivalents; covers note-takers + assistive technology + extra study time + quiet study spaces for tinnitus + hearing-loss-affected students.
- **Equality Act 2010 s. 6 (definition of disability)** — tinnitus + hearing-loss qualify as disability if substantial + long-term + adversely-affecting day-to-day activities; severity threshold is functional impact, not specific diagnosis. **Reasonable adjustments duty s. 20-22** is a *positive* duty on employers + service-providers + landlords + education-providers. Dossier readers in employment can request specific adjustments — quiet workspace, noise-cancellation, flexible-hours-for-sleep-debt-recovery, signed-language-interpreter, captioning, written-meeting-summaries.
- **Blue Badge + Disabled Person's Railcard** — secondary benefits.
- **NHS Continuing Healthcare + Personal Health Budgets** — for severely-disabled tinnitus + comorbid-MH cases.

### 5.3 AU — NDIS + Disability Employment Services + DDA reasonable adjustments

- **National Disability Insurance Scheme (NDIS)** — Commonwealth scheme administered by NDIA; eligibility for under-65s with permanent + significant disability. Tinnitus alone rarely meets the threshold but tinnitus + comorbid sensorineural-hearing-loss + comorbid-MH does. NDIS funds hearing aids + CI maintenance + sound-therapy devices + psychological-support + signed-language-interpreter. NDIS Quality and Safeguards Commission oversees.
- **Disability Employment Services (DES)** — DSS scheme placing disabled job-seekers + funding workplace modifications; tinnitus + hearing-loss-affected workers eligible.
- **Disability Discrimination Act 1992 ss. 5 + 24** — reasonable adjustments duty; AHRC conciliation + Federal Court enforcement.
- **Companion Card** — partner-/carer-access to events at no additional cost.

### 5.4 DE — Schwerbehindertenausweis + GdB + Inklusionsamt + Reha-Schmerztherapie

- **Schwerbehindertenausweis (severely-disabled ID card)** — issued by Versorgungsamt on Grad der Behinderung (GdB) ≥ 50; tinnitus alone usually GdB 0-20 (insufficient); tinnitus + sensorineural-hearing-loss bilateral ≥ 40 dB HL can reach GdB 30-40; tinnitus + bilateral profound hearing-loss + comorbid-MH can reach GdB ≥ 50. GdB 50 unlocks: tax allowances (Pauschbetrag § 33b EStG) + 5 additional vacation days + special protection from termination + parking concession (with aG-mark) + transport-fare reductions.
- **Inklusionsamt** — funds workplace accommodations + co-funds with employer for hearing-aids + assistive-listening + signed-language-interpreter (Gebärdensprachdolmetscher).
- **Reha-Schmerztherapie + Reha-Audiologie** — § 40 SGB V Krankenkasse-funded rehabilitation including multimodal tinnitus-rehab programmes (Universitätsklinikum Tübingen + Charité + Heidelberg + Wuppertal DTL-Schmerz-Rehabilitations-Klinikkette).
- **Behindertengleichstellungsgesetz (BGG) + AGG s. 19 service-discrimination** — the EHRC-equivalent positive-duty + protection-from-discrimination framework. ADS Beschwerde route.

### 5.5 IN — UDID + RPwD Act 2016 benefits + reservations + tax exemptions

- **Unique Disability ID (UDID)** — DEPwD-issued; required for accessing RPwD-Act benefits. Tinnitus + sensorineural-hearing-loss assessed under Hearing Impairment category (RPwD Schedule + 2018 Assessment Guidelines); benchmark disability ≥ 40% unlocks core benefits.
- **RPwD Act 2016 benefits** — § 32 reservation in higher education (5% reservation for benchmark disability); § 34 reservation in government employment (4% reservation for benchmark hearing-impairment + visual-impairment + locomotor-disability + multiple disabilities); § 33 identification of posts.
- **Income-tax exemption** — § 80U IT Act 1961 deduction (Rs 75,000 normal disability; Rs 1,25,000 severe disability); § 80DD for dependent disabled family-member.
- **ALIMCO** — Kanpur-based public-sector manufacturer providing subsidised hearing-aids + assistive-listening devices via state-government camp-distribution; ALIMCO Mukhya Mantri Camps in many states.
- **DEPwD scheme of Assistance for Disabled Persons** (ADIP scheme) — central-government subsidy for hearing-aids + assistive devices.
- **Composite Regional Centres (CRC)** — DEPwD-administered network for disability assessment + UDID issuance + assistive-device provision.

### 5.6 CA + EU + WHO-CRPD Article 28 frame

- **Canada Disability Benefit (CDB)** — federal benefit phased-in from July 2025 under Canada Disability Benefit Act 2023; means-tested supplement for 18-64 working-age disabled adults; eligibility tied to Disability Tax Credit (DTC).
- **EU EAA 2025** — workplace + e-commerce + service accessibility obligations; member-state implementation differs.
- **CRPD Article 28 (adequate standard of living + social protection)** — international floor; member-states obligated to non-regression + progressive realisation. Dossier can cite as the governing international framework.
