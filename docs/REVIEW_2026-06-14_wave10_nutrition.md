## 1. MACROS + KCAL + PROTEIN G/KG + AMDR + UNDER-RNI FLAGS FOR 65+ / PERIMENOPAUSAL / VEGAN / CHRONIC-UNDEREATER

### 1.1 What the dossier currently covers

§6.1 names "Mediterranean-MIND-Indian-millet base." §6.2 lists ~16 mandatory food rows with daily-gram targets. §6.4 covers sprouting + fermenting + bone broth + ghee-tempering. §6.5 lays out a 7-day meal plan by daily theme. §6.6 covers environmental triggers; §6.7 the cofactor floor.

**Zero kcal target. Zero protein g/kg. Zero AMDR. Zero under-RNI flag for 65+ sarcopenia, perimenopausal bone loss, vegan B12 / iron / Zn / EPA+DHA, or chronic-undereater female ferritin + Vit D combined deficit. Zero "what does a day's plate look like in calories" calculation.**

§6.5 calculates (by reviewer, not stated) at ~1,400-2,050 kcal across days, ~70-110 g protein. A 75-kg working-age male on Day 7 runs ~600 kcal deficit + ~0.95 g/kg protein (below ESPEN PROT-AGE 1.0-1.2 g/kg; well below the 1.2-1.5 g/kg 65+ sarcopenia target). A 55-kg perimenopausal female on Day 7 with skipped breakfast runs ~1,100 kcal + ~60 g protein — under-fuelled across BMD + lean-mass + low-mood + iron-replenishment cascades.

### 1.2 The five sub-populations the §6 silos

- **65+ sarcopenia + osteopenia / osteoporosis.** Wave-9 surfaced 45-55% of readers are 65+; protein PRI 0.83 g/kg (EFSA) is *known-inadequate* per ESPEN PROT-AGE 2014 + DGE 2024. Target: **1.0-1.2 g/kg healthy 65+; 1.2-1.5 g/kg sarcopenia + frail + recovery**. A 70-kg 75-year-old needs ~85-105 g protein/day, not ~60 g. The wave-9 fall arithmetic is *also* a calorie + protein arithmetic.
- **Perimenopausal female 40-55.** Estrogen-withdrawal phase: BMD loss ~1-2%/year + iron-replenishment-failure + visceral-adiposity-shift + low-mood overlap. Targets: ~1.0-1.2 g/kg protein, ~1,800-2,100 kcal at light-moderate activity, Ca 1,000-1,200 mg + Vit D 800-1,000 IU + Mg 320-360 mg + Vit K2 90 µg. §6.7 covers Vit D + Mg + K2 but not the perimenopausal sub-tier.
- **Vegan / strict-vegetarian.** §6.7 names vegan as B12-risk but no gates on iron, Zn, Ca, iodine, EPA+DHA (algal not flax — ALA→EPA ~5-8%; ALA→DHA ~0.5%), B2, selenium (soil-dependent), creatine (~50-70% of omnivore pool), choline (~half at egg-exclusion). Algal-DHA target dose: ~2,500-4,000 mg/day to hit ≥1 g DHA + ≥1 g EPA (Nordic Naturals Algae Omega 715, NOW Algal DHA 1,000, Testa DHA+EPA).
- **Chronic-undereater female 18-44.** ~15-22% of tinnitus-distress cohorts per Bauer + RNID + BTA service-user data. Chronic restriction → low Vit D + ferritin + Zn + EPA-DHA + K2 + B-vitamin cluster. The §6.5 plan assumes 1,800-2,000 kcal intake; a reader at 1,200-1,400 kcal can do every substitution and still hit every §6.7 deficiency. Calorie-floor gate missing.
- **Frail + lives-alone 80+.** Wave-9 Sub-gate C purges Combo 3 but has no parallel nutrition edit. Sarcopenia + dysphagia-screen + reduced appetite-regulation + social isolation → ~1,100-1,400 kcal/day actual; needs 1.2-1.5 g/kg protein + fortified mini-meals + ONS (Aymes Shake / Fortisip Compact Protein / Ensure Plus Advance).

### 1.3 The AMDR + macro split the §6 implies but never names

§6.5 implicit macro split (~40-45% carb / ~30-35% fat / ~20-25% protein) is consistent with EAT-Lancet 2019, PREDIMED 2018, MIND 2015. State explicitly at §6.1:

- **Carb 40-50% kcal** (≥30% low-glycaemic whole: millet, pulses, intact whole grain, greens, berries; ≤10% added/free sugar per WHO 2015).
- **Protein 20-25% kcal**, target **1.0-1.2 g/kg working-age; 1.2-1.5 g/kg 65+ sarcopenia / frail / perimenopausal-bone-prevention / recovery**.
- **Fat 30-40% kcal**, ≥10% MUFA (EVOO + avocado + nuts), 5-10% PUFA (≥2-3 g/day EPA+DHA with ≥1 g DHA; ω-6:ω-3 <4:1), <10% saturated, <1% trans.
- **Fibre 30-40 g/day** (SACN + DGE; soluble + insoluble; itself a tinnitus-distress + sleep + microbiome lever).
- **Sodium <2,000 mg/day** general; **<1,500 mg/day Ménière** (already in §11.6 Furstenberg-Bárány 2015).
- **Water 2.5-3 L/day; alcohol 0 g** (already in §6.2 + §6.3).

## 2. JURISDICTIONALLY-SPECIFIC DEFICIENCY-SCREENING CADENCE (B12 / VIT D / FOLATE / IRON / MG / ZN / HOMOCYSTEINE)

### 2.1 What the dossier currently covers

§11.3 names 17 comorbidity-marker rows at a blanket "baseline / month 3 / 6 / 12" cadence, almost all routed through paid DTC (LetsGetChecked, Thriva, Everlywell, Cerascreen, InsideTracker, Healthians, i-screen, Walk-In Lab, Doctor's Data, DUTCH, Lofta).

**Three structural gaps:**

1. **Blanket cadence.** B12 + folate + Vit D + ferritin recheck at month 3 is right; but TSH on stable euthyroid is annual not quarterly; Lp(a) is once-lifetime (genetic); hs-CRP single-reading is unreliable. Cadence must be marker-specific.
2. **Over-uses paid DTC, under-uses free public bloods.** UK NHS GP runs B12 + ferritin + 25-OH-D + TSH + lipid + U&E + HbA1c + LFT under SNOMED-coded indications (fatigue + tinnitus + dizziness qualify). German GKV Hausarzt §28-32 SGB V same. Australian Medicare MBS items 66536 / 66719 / 66848 / 66512 cover TSH / B12-folate / ferritin / lipid. Indian AIIMS / Apollo / KIMS / Manipal OPD ₹200-800/panel; PMJAY in-network. Only after free-public path fails does DTC make economic sense — §11.3 inverts this.
3. **No risk-tier stratification.** A 25-year-old omnivore-male B12 quarterly is over-screening; a 72-year-old PPI + metformin + vegetarian-female on baseline + annual under-screens. Perimenopausal heavy-menses ferritin biannual, not annual.

### 2.2 The marker-by-marker dietitian cadence ladder

- **B12 (serum + MMA + active-B12 / holo-TC).** Baseline + month 3 + month 12 + annually if supplementing; **quarterly first year if PPI / metformin / Crohn / pernicious anaemia / gastrectomy / vegan / 65+**. Holo-TC adds value when serum borderline (148-300 pmol/L).
- **Vit D 25-OH-D.** Baseline + month 3 + month 12; biannually high-risk + winter-end (Feb-Mar UK; Sep-Oct AU). Target **30-50 ng/mL (75-125 nmol/L)** per NHS + NICE + EFSA + DGE; **soften §6.7's 40-60 ng/mL** — >125 nmol/L raises hypercalcaemia + fractures per Bischoff-Ferrari 2025.
- **RBC folate** (preferred over serum, 90-120 day integration). Baseline + month 3 + month 12.
- **Ferritin + iron panel + TIBC + transferrin saturation.** Baseline + month 3 + 6; biannually if heavy menses / vegan / Crohn / coeliac / bariatric. Target >30 µg/L (WHO 2020); RLS + fatigue + tinnitus-distress often improve at >50-75 µg/L (Patrick + Trotti). Interpret with hs-CRP (acute-phase confounder).
- **RBC Mg** (more sensitive than serum). Baseline + month 3 + 12.
- **Plasma Zn:Cu** (target 8-15:1). Baseline + month 6.
- **Homocysteine.** Baseline + month 6 + annually. Target <8 µmol/L.
- **TSH + fT4 + anti-TPO.** Baseline + annually if stable euthyroid; quarterly if titrating. **fT3 + anti-Tg + rT3 are subspecialty add-ons, not routine — soften §11.3 from "as if routine".**
- **Lipid + ApoB annually; Lp(a) once-lifetime** (genetic; ESC 2024 + EAS).
- **Home BP.** 7-day average baseline + monthly × 3 + quarterly long-term. Daily home BP over-monitors + raises anxiety per BIHS + NICE + ESC.

### 2.3 Jurisdictionally-routed cadence ladder

**Tier 1 (try first) — Free public-system.** NHS GP under SNOMED indications (fatigue + tinnitus + dizziness qualify); GKV Hausarzt §28-32 SGB V routine; Medicare MBS items 66536/66719/66848/66512; AIIMS / Apollo / KIMS / Manipal OPD ₹200-800/panel + PMJAY in-network; Canadian provincial OHIP / RAMQ / MSP routine.

**Tier 2 — Pharmacy / community finger-prick** (Tier 1 slow): Boots + Superdrug £25-40; German Apotheke €15-25; AU Sonic Healthcare; India Healthians / Thyrocare home-draw ₹400-1,500.

**Tier 3 — DTC paid** (1 + 2 unavailable or combined panels needed): LetsGetChecked, Thriva, Everlywell, Cerascreen, InsideTracker, Doctor's Data, DUTCH, Lofta. £30-200/panel. Useful when a single panel (e.g., Thriva Active B12 + MMA + folate + ferritin £79) replaces NHS-GP separate-visit + 2-3 wk wait.

**Tier 4 — Specialist** (Endo / Haem / Nephro) when Tier 1-3 surfaces abnormality.

## 3. FODMAP + LOW-GLUTAMATE + LOW-HISTAMINE VARIANTS FOR SUB-TYPES

## 4. SUPPLEMENT-MEAL + CIRCADIAN TIMING CHRONOPHARMACOLOGY

### 4.1 What the dossier currently covers

§3.3 + §8.1-8.4 + §6.7 list dose, AM/midday/PM placement, antagonist class. **What is missing is the timing-specificity fork:** with food vs fasting, AM vs PM, paired-with-X vs separated-from-Y by ≥2 h, circadian phase.

### 4.2 Eight high-leverage timing rules invisible in §3 + §6.7 + §8

- **Mg form-dependent.** Glycinate: with food, evening (~300-400 mg). L-threonate: **empty-stomach evening 60-90 min pre-bed** (~1,000 mg) — with food blunts CNS bioavailability per Slutsky 2010 + Liu 2016. Citrate / oxide: morning + lunch with food — evening laxative-shift = bedside-bathroom-trip risk in 65+ (wave-9 falls arithmetic). §3 + §6.7 + §8.4 list dose but never form-vs-timing fork.
- **B12 morning, away from food** (avoids R-protein binding; aligns with cortisol awakening). §3 + §6.7 + §8.1 unspecified.
- **Iron + tea/coffee/Ca/Zn → ≤50% absorption** (polyphenol-chelation per Hurrell 1999 + Disler 1975). 2-h gap mandatory. **Empty-stomach morning + Vit C 100-250 mg; alternate-day 60 mg outperforms daily 60 mg** per Stoffel 2017/2020 (hepcidin-suppression cycle).
- **Vit D + K2 with the day's largest fat meal** — absorption ~32% higher per Mulligan 2010. §6.7 says "with fat" but no timing fork.
- **Zn evening with Cu-rich meal** (oysters / liver / cashew / sesame / dark chocolate already in §6.2) OR paired Cu 1-2 mg supplement; Zn ≥2 h from Fe + Ca (DMT-1 competition); Zn alone >8 weeks suppresses Cu → cytopenia + myelopathy (wave-3 supplement-QC).
- **Ashwagandha with light snack not empty-stomach.** KSM-66 + Sensoril cause nausea / GI upset in 8-12% per Choudhary 2017 + Lopresti 2019. §8.4 Combo 3 21:00 empty-stomach in wave-9 65+ sub-tier B + C raises dyspepsia-driven sleep fragmentation — opposite of intent.
- **Ginkgo (non-anticoagulated only) split-dose breakfast + lunch**, not evening (sleep activation in ~5-8%); split-dose > single 240 mg per Schwabe PK.
- **Curcumin timing-fork by formulation:** Meriva 2×/day with meals; C3 Complex 2×/day with meal + piperine; Longvida 1×/day. §3.3 + §6.2 + §6.4 already cover Shoba 1998 piperine + lipid-vehicle.

### 4.3 Specific edits

- **Edit 4.1.** Add **§3.5 "Supplement-meal + circadian timing table"** — 12 OTC molecules × meal fork (food / fasting / fat-meal / empty-stomach) × circadian phase × interaction (≥2 h gaps) × side-effect mitigation.
- **Edit 4.2.** Update **§6.7 Mg row** — fork glycinate (with food evening) / L-threonate (empty-stomach evening 60-90 min pre-bed) / citrate-oxide (morning + lunch; not evening).
- **Edit 4.3.** Update **§6.7 iron row** — alternate-day per Stoffel 2017; morning empty-stomach or with Vit C; ≥2 h from tea/Ca/Zn.
- **Edit 4.4.** Update **§6.7 Vit D + K2 row** — take with day's largest fat meal.
- **Edit 4.5.** Update **§8.4 Combo 3** — Ashwagandha with light snack, not empty-stomach (8-12% nausea sub-cohort).
- **Edit 4.6.** Update **§3.3 Ginkgo row** — non-anticoagulated only: split-dose breakfast + lunch, avoid evening.

### 4.4 Verdict on Dimension 4

**Moderate revision.** §3 + §6.7 + §8 list dose without timing-fork; adherence + efficacy + side-effects all timing-sensitive. ~600 words additive + §3.5 table + 5 row-level updates. Doctrine intact — Mulligan 2010 + Stoffel 2017/2020 + Slutsky 2010 / Liu 2016 + Choudhary 2017 / Lopresti 2019 + Shoba 1998 as evidence-anchors.
