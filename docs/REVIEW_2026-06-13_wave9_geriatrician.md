## 1. PRESBYCUSIS + AGE-RELATED HEARING LOSS — THE UPSTREAM LEVER SILOED AS A DOWNSTREAM ADJUNCT

### 1.1 What the dossier currently covers

§2.6 names "tonal-hearing-loss-driven" as one of seven sub-types at ~40-60% of presentations. §5.9 (wave-3 disability-inclusion + ENT-audiology) covers Widex Zen / Phonak Tinnitus Balance / Oticon SoundSupport / ReSound Tinnitus Relief / Starkey Multiflex / Signia Notch Therapy + Apple AirPods Pro 2 + the US-only OTC class. §11.3 has a "hearing loss" comorbidity row. §8.0 MVP places sound enrichment as layer (1) but **does not name hearing-aid trial as the frontline 65+ step**. §1.x Executive Summary does not name presbycusis. §2.2 names "aging" generically.

### 1.2 The geriatric inversion

In 65+, **tinnitus is presbycusis-and-its-sequelae until proven otherwise**. UK BSA + ISAAR + DGGG / DEGAM + ICMR + AIIMS NCC + NHMRC + Hearing Australia + Audiology Australia converge on ~70-80% of new 65+ tinnitus being primarily presbycusis-driven (often with mild noise-exposure history). Pattern: bilateral symmetric high-frequency sloping SNHL, 4-8 kHz notch superimposed. Schuknecht's four sub-types + slow-progression prognosis are mechanistically distinct from the noise-injury sub-type the dossier's working-age framing centres.

For a working-age noise-injury patient, supplements (§3) + sound (§5.2) + MBCT-T (§5.5) are frontline; HA is §5.9 downstream. For a 65+ presbycusis patient, **hearing intervention is frontline + highest-leverage single intervention**; supplements + sound + MBCT-T are downstream adjuncts. The clinical inversion is total.

### 1.3 The hearing-intervention ladder for 65+ — never laid out

Wave-3 built §5.9 at device-list level but did not anchor the stepped 65+ ladder:

- **Step 1.** Clinical audiogram + tympanometry + bone-conduction + acoustic reflex (smartphone audiograms miss mid-frequency notch 30-50%, worse in 65+ complexity). NHS Audiology UK + HSP + GKV §28-32 SGB V + ICMR-AIIMS cover free/near-free for 65+.
- **Step 2.** Mild-to-moderate symmetric loss → bilateral HAs with tinnitus-masker programs (clinic-grade ~$2-5k; NHS-fitted free; GKV Festbetrag €784.94/ear per wave-6; HSP free for pensioners + DVA).
- **Step 3.** Severe-to-profound bilateral loss → CI referral. Geriatric CI in 65+ is under-recognised highest-leverage: Lenarz 2012 + Mosnier 2015 + Roberts 2013 show ~80% of 65-85 recipients report meaningful tinnitus reduction alongside hearing restoration. Dossier names CI exclusions for §5 devices but never CI as a 65+ presbycusis-progression option.
- **Step 4.** Asymmetric severe loss → CROS / BiCROS or single-side CI. Zero CROS coverage.
- **Step 5.** Mixed-conductive (otosclerosis + chronic OME + conductive presbycusis) → BAHA / Ponto / middle-ear implant. Dossier excludes from §5 but never names as positive 65+ option.

### 1.4 Specific edits

- **Edit 1.1.** Add **§2.6 sub-type row 8 — Presbycusis-dominant tinnitus (65+)**: bilateral symmetric high-frequency sloping audiogram, slowly-progressive, 70-80% of new 65+ presentations; first-line hearing intervention + sound + MBCT-T; supplements adjunctive.
- **Edit 1.2.** Update **§1.x Executive Summary** — add 2-sentence 65+ paragraph: highest-leverage intervention is §5.9 hearing intervention (not §3 supplements); start with audiogram + hearing-intervention decision before supplement stack.
- **Edit 1.3.** Elevate **§5.9 to §5.0 frontline-for-65+** — add 5-step ladder (audiogram → HA → CI → CROS → BAHA); name NHS Audiology + GKV §28-32 SGB V + HSP + AIIMS + Ayushman HCSCC.
- **Edit 1.4.** Update **§8.0 MVP layer (1) sound enrichment** — add 65+ qualification: layer (1) is hearing-intervention-first; sound enrichment is layered on top of a fitted HA, not instead of one.
- **Edit 1.5.** Add **§11.3 CI-candidacy row**: bilateral severe-to-profound SNHL (PTA >70 dB; word-recognition <60% best-aided) + 65+ + tinnitus-distress refractory to HA → CI assessment (NHS BCIG + GKV CI-clinics + Hearing Australia + ICMR-AIIMS + Cochlear Foundation India subsidy).

### 1.5 Verdict on Dimension 1

**Major revision.** Geriatric inversion places hearing intervention as upstream frontline + supplements as downstream adjuncts. ~700 words additive inverts clinical sequencing for ~45-55% of readers. Doctrine intact — HAs + CROS + BAHA + CI remain OTC-or-clinic-route.

## 2. POLYPHARMACY 65+ — STOPP/START + BEERS + EU(7)-PIM + FORTA + ACB OPERATIONALISATION

### 2.1 What the dossier currently covers

§10.1 lists 22 drug-class rows (loop diuretics, aminoglycosides, NSAIDs/aspirin, anti-seizure, tricyclics, SSRI/SNRI, benzos/Z-drugs, betahistine, steroids, erection pills, warfarin/DOAC, antiplatelets, BP drugs, β-blockers, statins, thyroid, metformin/PPI, anti-cholinergic OTC sleep, first-gen antihistamines, lithium, quinine, plus wave-8 ACE-i + cardioprotective-aspirin caveat). §10.5.5 covers 6 real-clinic AE patterns + 4 PV signals. §11.5 elderly+polypharmacy row says "halve melatonin + halve Ashwagandha + B12 CBC monitor." Cardiology-comprehensive after wave-8 but **geriatric-PIM-blind**.

### 2.2 The five geriatric PIM frames missing

- **STOPP/START v3 (O'Mahony 2023)** — 191 STOPP + 35 START criteria. Embedded in NHS + RCP + DGG + Irish HSE + ANZSGM + ICMR. Directly relevant: STOPP B7 (loop diuretic for ankle oedema without HF), D11 (long-term benzo), K1-K4 (anti-cholinergic burden = falls), START A6 (statin in post-MI 65+).
- **Beers Criteria 2023 (AGS)** — US-derived → cited as scientific-anchor not regulator-authority (same as wave-3 AAO-HNSF). 30+ PIMs: first-gen antihistamines, chronic benzos + Z-drugs, anti-cholinergic burden, chronic NSAIDs, chronic PPI >8 weeks.
- **EU(7)-PIM list (Renom-Guiteras 2015 + DGGG consensus)** — 282-PIM list. Adds Vinpocetine (already wave-2 corrected for AU/CA/US), some CCBs, doxazosin first-line HTN, chronic glibenclamide.
- **FORTA (Wehling 2008 Heidelberg)** — A/B/C/D classification for 65+. Operationalised in DGG + DEGAM + GKV. §10.1 supplements map: Mg = A; melatonin = B; Ashwagandha = C in 75+ + polypharmacy; valerian = C; St John's Wort = D in 65+ on warfarin/SSRI.
- **Anti-cholinergic burden score (ACB — Boustani / Carnahan; Salahudeen 2015)** — Cumulative ACB >3 ≈ 1.5× falls + 1.5× cognitive impairment + 1.5× delirium + 1.5× all-cause mortality (Fox 2011; Richardson 2018; Cardwell 2015). §10.1 surfaces individual drugs but **never computes cumulative burden** — a 65+ patient on amitriptyline (+3) + oxybutynin (+3) + chronic diphenhydramine (+3) sits at ACB ≥9, well above the 3-point threshold.

### 2.3 65+-specific overlays on Mg + warfarin + statin + diuretic + antihypertensive

Wave-8 added loop-diuretic dose-rate + intra-class hierarchy + ACE-i bradykinin tinnitus + statin auditory-cluster + cardioprotective aspirin. Geriatric overlays the cardiology layer does not name:

- **Mg + warfarin + age-related eGFR decline.** Cockcroft-Gault eGFR drops ~1 mL/min/year after age 40; a 75-year-old with serum creatinine 90 µmol/L may have eGFR 50 (CKD 3a) despite "normal" creatinine. §11.3 eGFR table needs an **age-tier overlay**: any reader 75+ → Mg halved unless eGFR verified >60. STOPP B9 flags this.
- **Statins + 65+ deprescribing.** 65+ deprescribing governed by life-expectancy + secondary-prevention indication, not auditory signal alone. STOPP A6 + Beers: continue in post-MI / post-stroke / FH + life-expectancy >1-2 years; consider discontinuation in primary-prevention 75+ with limited life expectancy. Auditory signal is tiebreaker.
- **Diuretics in 65+ — STOPP B7 the dossier never names.** Loop diuretic for ankle oedema without clinical HF is a top-5 STOPP item — common in 65+ on long-term GP-prescribed furosemide for "swelling" without echo confirmation. If no HF, deprescribing is the answer + tinnitus aggravator removed simultaneously.
- **Antihypertensives in 65+ — orthostatic-hypotension + falls balance.** STRIDE-BP + SPRINT-Senior + HYVET show frail 75+ targeted to <130 systolic experience increased orthostatic-hypotension + falls; cochlear-perfusion benefit (wave-8 Dim 4) is balanced against hip-fracture mortality. Reader needs orthostatic-hypotension home check (lie 5 min → stand 1-min + 3-min BP; >20/10 mmHg drop = positive) before targeting <130/80.
- **§8.5 06:00 "pinch of salt + lemon" + §6.3 low-salt inversion in 75+ on diuretic + antihypertensive.** Wave-8 carved out HFrEF; geriatric overlay extends to broader frail-75+ on antihypertensives. Morning pinch-of-salt is geriatrically correct for orthostatic-hypotension prevention; §6.3 should carve out the 75+ sub-cohort.

### 2.4 Specific edits

- **Edit 2.1.** Add **§10.1.6 "STOPP/START + Beers + EU(7)-PIM + FORTA + ACB — operationalising §10.1 for 65+"** ~500 words: the five frames; self-administered ACB calculation worksheet (list every drug + supplement + OTC; assign 0/1/2/3 per Boustani; sum; >3 = deprescribing target); STOPP/START matched to §10.1 rows (B7, K1-K4, D11, A6); FORTA A/B/C/D categorisation of dossier supplements.

## 3. FALLS-RISK ON §8 COMBO 3 — TRIPLE-SEDATION CLUSTER NONE OF THE 38 PRIOR REVIEWERS QUANTIFIED

### 3.1 What the dossier currently covers

§8.4 Combo 3: **Mg L-threonate 1 g + Ashwagandha KSM-66 300 + Melatonin 0.3-1 mg + optional Saffron 28 mg (perimenopausal) — 3-4 caps at 21:00.** §8.5 adds Ashwagandha "≥1.5 h pre-bed" + Yoga Nidra 21:30 + 4-7-8 + bedside masker 21:45 + bedtime 22:00-22:30. §10.4 covers individual sedative-additive risk. §10.5.5 surfaces "Mg L-threonate + melatonin → vivid dreams / fragmented REM ~24%" but not falls. §11.5 elderly+polypharmacy row: "Re-evaluate all sedating supplements (Mg + melatonin + Ashwagandha + valerian = falls risk); halve melatonin to 0.1-0.3 mg; halve Ashwagandha." Wave-4 + wave-5 flagged framing; **none quantified the fall + hip-fracture arithmetic**.

### 3.2 The triple-sedation arithmetic the dossier never computes

In geriatric pharmacology, the **sedation cluster + nocturia + ambient hypotension at first-rise** is the dominant nocturnal-fall mechanism in 65+ (Sterke 2012; Park 2015; ANZSGM Falls Position 2022; BGS Multifactorial Falls Assessment 2023; ICMR cohort). §8.4 Combo 3 stacks three sedative effects:

- **Mg L-threonate 1 g** — NMDA + GABA-A potentiation; offset 6-8 h; in 65+ with impaired renal clearance (Dim 2), effective concentration elevated. Liu 2016 (Tsinghua) trial was working-age adults.
- **Ashwagandha KSM-66 300** — withanolide GABA-mimetic + mild sedative; wave-2 hepatotoxicity + TGA 2024 CNS cluster 65+-amplified.
- **Melatonin 0.3-1 mg** — direct circadian + mild sedative; in 65+, age-related endogenous melatonin decline amplifies dose effect; >0.3 mg approaches pharmacological dose with morning-grogginess + nocturnal-fall amplification (Lyseng-Williamson 2012 Circadin geriatric review).
- **Optional Saffron 28 mg** — serotonergic; mild sedative.

The triple stack layered onto a 75+ patient who (a) is also on a benzo, Z-drug, or trazodone (ACB +3, FORTA D), (b) has sarcopenia + osteopenia/osteoporosis, (c) needs to rise 1-2× overnight for nocturia (BPH + diuretic-induced + age-related), (d) navigates a partially-lit unfamiliar room with rugs / cords / stairs, faces a **measurable fall + hip-fracture arithmetic**. UK NHFD + German EPOS-Register + Australian ANZHFR + AIIMS hip-fracture cohort converge: hip-fracture 1-year mortality ~20-25%. §11.5's "halve" is geriatrically directionally correct **but under-specifies harm magnitude + ambulatory-status gate** that drives purge-vs-halve-vs-defer.

### 3.3 Ambulatory-status sub-gates the §11.5 row needs

- **Sub-gate A — Robust 65-74**, no fall history, independent mobility, no walking aid, ≤2 chronic drugs: Combo 3 half-dose acceptable (Mg L-threonate 500 mg + Ashwagandha 150 + melatonin 0.1-0.3 mg); skip saffron.
- **Sub-gate B — Pre-frail 75-84**, no fall history, polypharmacy 3-5 chronic drugs: Combo 3 reduced to single layer (melatonin 0.1-0.3 OR Ashwagandha 150 OR Mg L-threonate 500, not three); add night-light + clear-path + bedside commode if nocturia frequent.
- **Sub-gate C — Frail 75+**, any fall in past 12 months, walking aid, polypharmacy >5 chronic drugs, lives alone: **Purge Combo 3 entirely.** Substitute Mg glycinate 200 mg at dinner not 21:00 + sleep hygiene + bedside sound machine + MBCT-T audio + multifactorial CGA-FaR (NHS BGS FaRTSS; ANZSGM Stay-Independent; AIHW Falls Prevention; ICMR; BFP Germany). Add night-light + clear-path + bedside commode + grab rails + non-slip footwear + DXA + FRAX + bisphosphonate / denosumab discussion with GP.

### 3.4 §8.5 evening sequence is also fall-amplifying in 65+

§8.5 21:00 Combo 3 → 21:30 Yoga Nidra + 4-7-8 → 21:45 bedside masker + side-sleep → 22:00-22:30 lights-out. Calibrated for working-age adults. Geriatric overlay: **21:30 Yoga Nidra supine** — risk of rolling-off + difficulty rising; chair/bed-based variant. **21:45 side-sleep affected-ear-up** — may conflict with osteoporosis + shoulder-pain + rotator-cuff degeneration in 75+. **Nocturia rise at 02:30** — never addressed; sedation + ambient hypotension + dim lighting = highest-fall-density hour per ANZHFR + NHFD. **22:00-22:30 blackout** — blackout in frail 75+ home is documented fall accelerator; hallway night-light is geriatrically protective.

### 3.5 Specific edits

- **Edit 3.1.** Add **§11.5 "Frail / 80+ / falls-history" row**: Combo 3 PURGED; substitute Mg glycinate 200 mg at dinner; CGA-FaR + DXA + FRAX + bisphosphonate / denosumab precede §3 + §5 + §7; STOPP K1-K4 deprescribing; night-light + clear-path + bedside commode + grab rails + non-slip footwear baseline.
- **Edit 3.2.** Update **§11.5 "Elderly > 65 + polypharmacy" row** — add three sub-gates A/B/C; name fall + hip-fracture arithmetic.

## 4. COGNITIVE DECLINE + DEMENTIA COMORBIDITY VS MBCT-T + SELF-SCORING DEMAND FLOOR

### 4.1 What the dossier currently covers

### 4.2 The MCI + early-dementia comorbidity the dossier silos

65+ tinnitus sub-population: MCI ~15-20% (Petersen 2014 Mayo; Brodaty 2013 Sydney; AIIMS NCC; DEGAM). 75+: MCI + early-dementia combined ~25-30% (Livingston 2020 Lancet Commission; ADI 2023; UK CFAS II + III; AIIMS LASI 2021). **Hearing-loss + dementia bidirectional** — Livingston 2020 names hearing loss as the single largest modifiable midlife dementia risk factor (PAF 8%); tinnitus + presbycusis + cognitive change cluster via shared sensory-cortical mechanisms + social-isolation + reduced cognitive reserve.

§5.5 MBCT-T self-applied + §11.2 self-scoring stack assume working memory + sustained attention for McKenna 2017's 8-week curriculum, executive function for daily 30-min meditation + daily slider + monthly battery, metacognitive monitoring, literacy + reading endurance for ~150-page workbook + MCID interpretation, and technology literacy for Bearable / Cronometer / smartphone audiogram / HRV trainers. In MCI + early-dementia, **all five are below threshold**. A 78-year-old with Mini-ACE 75/100 attempting six instruments at month 6 + month 12 will either disengage (wave-4 adherence failure) or self-score with instrumentation noise larger than the MCID.

### 4.3 What the geriatric framing needs

- **Cognitive-screen baseline.** MoCA (Nasreddine 2005 — MCI sensitivity ≤26); Mini-ACE (Hsieh 2015 — quick + IP-free); GPCOG (Brodaty 2002 — primary-care-validated free); MMSE (Folstein 1975 — known but less MCI-sensitive). Add **§11.3 row "Cognitive screen"** for 65+: free at GP / Hausarzt / Indian primary-care / Australian Cognitive Decline Partnership Centre; self-administrable versions exist (Mini-ACE app; GPCOG online).
- **Cognitive-demand floor for §5.5.** MoCA <23 / Mini-ACE <70 / GPCOG <8 → substitute caregiver-mediated MBCT-T OR clinician-mediated MBCT-T (NHS IAPT-Older-Adults; German Psychotherapie §31a/§32 SGB V; Australian Better Access; AIIMS NCC geriatric psychiatry).
- **Caregiver-mediated scoring pathway.** Caregiver reads questions + records responses + interprets MCID; cadence quarterly not monthly.
- **Reduced-cognitive-demand instrument selection.** For MCI + early-dementia, reduce from 7 instruments to **TFI + 0-10 slider + PHQ-9 only**.
- **Delirium-overlap caveat.** 75+ acutely hospitalised or post-anaesthesia: delirium + new tinnitus + new hearing loss is a clinical cluster (CAM + 4AT + NICE delirium + DGNI + DGGG). Name as **defer-protocol** state.
- **Hearing-loss + dementia bidirectional flag.** Lin 2023 ACHIEVE RCT showed 48% slower cognitive decline in 3 years with HAs in at-risk 70-84 cohort; §5.9 (elevated per Dim 1) should name dementia-risk-reduction as co-benefit.

- **Edit 4.2.** Add **§11.6 sub-bullet "MCI or early dementia"** (MoCA <23 / Mini-ACE <70 / GPCOG <8): defer §5.5 self-applied; substitute caregiver-mediated OR clinician-mediated via NHS IAPT-Older-Adults / German §31a SGB V / Australian Better Access / AIIMS NCC; reduce §11.2 to TFI + 0-10 + PHQ-9; quarterly cadence.
